Hello!
Welcome to the solar system simulator. This is a python program that runs using the pygame library. It is a simulator game that simulates the physics of a solar system created by the user, with the ability to move around and place any objects you want to.

To run the program:
1. Download the files from the repository
2. Make sure you have Visual Studio Code installed - the program will run in the VS Code IDE
3. Make sure you have python installed - Python v.3.9.0 and above versions should work
4. Make sure you have the necessary python libraries installed (list is below)
5. Open program files in VS Code, and make sure the correct python interpreter for your version is selected
6. Run the program and enjoy!

List of python libraries needed for this program:
1. pygame (if pygame doesn't work, install pygame-ce for newer versions of python)
2. math
3. numpy
4. matplotlib.pyplot
5. random
6. sys
